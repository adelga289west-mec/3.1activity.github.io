Difference between datalist element and select element

The difference between datalist and select is that in datalist, one is still able to type in their own value while in select you can only choose from the given value.

Can you change input values if they are inside a disabled fieldset?

What are the 6 main attributes of the meter tag?

The 6 main attributes of the meter tag are high, low, min, max, optimum, and value.
